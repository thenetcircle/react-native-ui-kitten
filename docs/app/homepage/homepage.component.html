WHAT IS IT?

React Native UI Kitten is a framework that contains a set of commonly used UI components styled in a similar way. You focus on business logic and it takes care of visual appearance. We have brought the concept similar to css classes that will save your time when styling some particular elements. And the most awesome thing: you can change themes on the fly by just passing a different set of variables. 100% native. Give our kitten a try!
